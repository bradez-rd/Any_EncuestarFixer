## Mini tutorial para usarlo en Chrome 

1. **Instala Tampermonkey en Chrome**  
   Ve a la Chrome Web Store, busca **Tampermonkey** y agrégalo al navegador.

2. **Entra a la página donde quieres usarlo**  
   Abre la web compatible y espera a que aparezca el botón o enlace de **Completar Encuesta**.

3. **Crea un nuevo script**  
   Abre Tampermonkey, pulsa **Create a new script** y borra el contenido que aparece por defecto.

4. **Pega el código del archivo `Any_EncuestarFixer.js`**  
   Copia todo el contenido del script y guárdalo con **Ctrl + S**.

5. **Ejecuta la herramienta**  
   Cuando aparezca la interfaz, elige el modo que quieras:

   - BAD  
   - REGULAR  
   - GOOD  
   - VERY GOOD  
   - EXCELLENT  
   - RANDOM  

   Luego pulsa **START**.

6. **Opcional**  
   Edita el script para otros sitios.  
   Pronto más compatibilidades. 😎
